# AI Agent 记忆系统 研究报告 | 2026-04-25

## 核心发现（5 条）

### 1. AI Memory 4W 综述：统一问题空间
2026年4月最新综述论文提出4W分类框架（What/When/Why/How），将零散的记忆研究统一为可对齐讨论的问题空间。论文还定义了三层边界（内容/存储/模态），提出多智能体记忆机制与四维评估标准。
- 来源：CSDN 博客解读（2026-04-20），原文 arXiv 2025年底发布

### 2. 六大开源记忆框架深度横评（Mem0/Zep/LangMem/MemOS/TiMem）
2026年4月横评文章对比了五大框架的架构差异：
- **Mem0**：KV存储+语义向量索引，极简架构，延迟低，v2 API支持memory export，官方MCP Server已上线
- **Zep**：长期记忆即服务，Postgres关系存储+语义检索
- **LangMem**：LangChain生态记忆层
- **MemOS**：记忆操作系统，统一管理异构记忆
- **TiMem**：来自arXiv:2601.02845，创新性的时序感知记忆机制
- 来源：CSDN（2026-04-03）

### 3. Agentic Memory 范式：多Agent推理替代向量检索
完全抛弃向量数据库和Embedding，用多个专业化Agent并行阅读和推理已存储的信息，以"认知理解"替代"数学相似度"。代表系统ASMR（Supermemory），编排多个搜索Agent分别处理直接事实、上下文线索、时间线重建。
- 来源：Agent Memory综述文章（2026-04-18）

### 4. 文件系统记忆方案在基准测试中表现优异
Letta基准测试显示，纯文件系统方案在LoCoMo上达到74.0%，击败了多个专业记忆系统。OpenClaw采用文件+向量混合方案，兼具透明性和检索能力。
- 来源：Agent Memory综述文章（2026-04-18）

### 5. 记忆系统五大设计决策的核心权衡
1. **向量检索 vs Agent推理检索**：语义相似≠语义正确
2. **集中式 vs 分布式记忆**：统一管理 vs 上下文隔离
3. **被动存储 vs 主动提取**：存取分离 vs 认知驱动
4. **精确召回 vs 语义模糊匹配**：事实准确 vs 灵活理解
5. **隐私与部署模式**：本地优先 vs 云端便利
- 来源：Agent Memory综述文章（2026-04-18）

## 知识图谱变化

- 新概念：4W分类、Agentic Memory、TiMem时序记忆、三层边界模型
- 新关系：4W→分类体系，Agentic Memory→多Agent推理范式，TiMem→时序感知
- 已验证方法：文件系统+向量混合架构在LoCoMo基准表现最优

## 学习记录

- 纯向量检索的语义相似陷阱（"住北京"vs"搬到上海"语义相似但含义相反）→ 需要时序推理能力
- Agentic Memory是2026年最前沿方向，但成本较高
- Mem0 MCP Server上线意味着记忆服务正在成为Agent基础设施标准组件
