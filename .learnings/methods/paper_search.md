# 论文搜索方法

> 来源：实践验证
> 添加时间：2026-04-07

## 步骤

### 1. 关键词策略
- **广泛搜索**：使用 1-2 个核心关键词，如 "agent memory"
- **精确搜索**：添加限定词，如 "agent memory architecture LLM 2024"
- **组合搜索**：使用引号精确匹配，如 `"knowledge graph memory" agent`

### 2. 数据源优先级
1. arXiv cs.AI, cs.CL, cs.LG 最新
2. Papers with Code（按引用排序）
3. Google Scholar（引用关系）

### 3. 过滤标准
- 发表时间：最近 90 天优先
- 引用数：>10 或 arXiv 新论文
- 来源：顶级会议优先（NeurIPS/ICML/ICLR）

## 效果
- 成功率：已验证 3 次
- 适用场景：新领域探索
